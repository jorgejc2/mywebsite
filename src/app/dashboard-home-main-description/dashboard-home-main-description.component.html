About Me

Taking AP Computer Science Principles in high school is what peaked my interest in programming and led me to pursue computer engineering. I was more enticed by the software aspect and seeing how much of a creative outlet it gave people. I remember my first dream was creating a gaming app for the app store and having it be trending.

My only issue at the time was that I had never coded in my life and felt I wasn’t capable and didn’t have the knowledge to do so.

Despite that, I applied everywhere and thankfully got accepted to U of I.

The courses I’ve taken have only empowered me and helped me grow in confidence with my ability to code and work with hardware.

I’ve grown a deep appreciation for computers and the impact they can have around us.

{{ project_names[curr_project] }}

During the summer of 2022, I had a chance to tackle a project involved with embedded systems as well as digital signal processing. It was a very involved experience and it led me to appreciate digital signal processing a lot more, a field that I previously didn't grasp as well.

By working with an Adalm Pluto SDR Learning Module and a software defined radio, I was able to apply digital signal processing to real world signals and thus, gained invaluable experience that gave me a good knowledgeable foundation of the topic. Due to hardware limitations of the software defined radio that I was programming, I had to seek creative measures to preserve memory while still being able to use this radio to listen to radio frequencies for hours on end and characterize its findings.

I found the project to be so much fun that I ended up buying my own personal Adalm Pluto SDR and creating scripts for it that used its full capabilities. This link is to my github page that contains scripts that can be run on a Pluto and has instructions on how to set one up.
